## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline pill padding (px-2 py-1) |
| sm | 8px | Button icon-to-label gap, intra-card row spacing |
| md | 16px | Card internal padding, section gap |
| lg | 24px | Grid column gap, modal section padding |
| xl | 32px | Page horizontal padding, modal outer padding |
| 2xl | 48px | Empty state vertical padding top/bottom |
| 3xl | 64px | Page-level vertical breathing room |

- Touch targets for LikeButton and BookmarkButton: minimum 44px height (accessibility — mobile tap area)

## Interaction Contracts

### 갤러리 카드 (GalleryCard)

- Default state: `bg-[#1A1A1E] border border-white/5 rounded-2xl p-4`
- Hover state: `hover:border-[#B48C50]/30 transition-all` (200ms ease — 기존 ProjectCard 패턴)
- Cursor: `cursor-pointer`
- Click: 모달 열기 (GalleryModal)
- Skeleton: `bg-[#4A4440] animate-pulse rounded-xl` (pulse 애니메이션, 200ms fadeIn 후)

### 좋아요 버튼 (LikeButton)

- Unfilled: `Heart` icon (lucide-react), `text-[#A09890]`, size 16
- Filled (liked): `Heart` icon with `fill-[#B48C50] text-[#B48C50]`
- Count: 14px weight 400 `text-[#A09890]` 버튼 우측
- Toggle animation: opacity + scale 100→110→100, duration 150ms
- Touch target: min-height 44px, min-width 44px (padding compensates icon size)
- Optimistic update: 클릭 즉시 로컬 상태 반영, API 실패 시 롤백
- Accessibility: `aria-label="좋아요"` (icon-only — screen reader must identify action)

### 북마크 버튼 (BookmarkButton)

- Unfilled: `Bookmark` icon, `text-[#A09890]`, size 16
- Filled (saved): `Bookmark` icon with `fill-[#B48C50] text-[#B48C50]`
- Touch target: min-height 44px, min-width 44px
- Optimistic update: 좋아요와 동일 패턴
- Accessibility: `aria-label="북마크"` (icon-only — screen reader must identify action)

### 정렬 탭 (GallerySortTabs)

- Tab container: `flex gap-0 border-b border-[#4A4440]`
- Inactive tab: `px-4 py-2 text-[14px] text-[#A09890] cursor-pointer`
- Active tab: `px-4 py-2 text-[14px] text-[#B48C50] border-b-2 border-[#B48C50] cursor-default`
- Transition: `transition-colors duration-150`
- Labels: "최신순" | "인기순"

### 모달 (GalleryModal)

- Backdrop: `fixed inset-0 bg-black/60` — 클릭 시 닫힘
- Container: `bg-[#1A1A1E] border border-[#4A4440] rounded-2xl max-w-[720px] w-full mx-4`
- Entry animation: `animate-fadeIn` (기존 200ms ease forwards — index.css 정의됨)
- Close button: X icon (lucide-react `X`), `text-[#A09890] hover:text-[#E8E2DA]`, top-right corner, `aria-label="닫기"`
- Scroll: 모달 내부 overflow-y-auto, max-height 80vh
- ESC 키: 모달 닫힘 (keydown 이벤트)

### 발행 버튼 (PublishButton in ProjectCard)

- Unpublished state: `"발행"` text, `bg-[#B48C50]/10 text-[#B48C50] border border-[#B48C50]/30 rounded-lg px-3 py-2 text-[13px]`
- Published (공개중) state: `"공개중"` badge + `"철회"` text button, `text-[#A09890]`
- 철회 hover: `hover:text-red-400 transition-colors`

### 무한 스크롤

- Sentinel element: `div` at bottom of GalleryGrid, observed by IntersectionObserver
- Loading indicator: GallerySkeletonCard 3개 렌더 (2칸/3칸 그리드 유지)
- Batch size: 12 cards per page (12 ÷ 3칸 = 4행 — 시각적으로 완결)
- End of feed: "모든 프로젝트를 불러왔습니다" — 14px text-[#A09890] text-center py-8

## Layout Contract

### GalleryPage (/gallery)

```
[GalleryPage]
  [PageHeader]          — "브랜드 갤러리" (Display 20px semibold)
  [GallerySortTabs]     — 최신순 | 인기순
  [GalleryGrid]         — CSS grid 반응형
    [GalleryCard × N]
    [GallerySkeletonCard × 3] (로딩 중일 때)
  [Sentinel div]        — IntersectionObserver target
  [EmptyState]          — 공개 프로젝트 없을 때
```

Grid breakpoints:
- Mobile (< 768px): `grid-cols-1` — 1열
- Tablet (768px~1023px): `grid-cols-2` — 2열
- Desktop (≥ 1024px): `grid-cols-3` — 3열
- Gap: 16px (md) mobile, 24px (lg) desktop

### GalleryCard 내부 구조

```
[Card container]  rounded-2xl bg-[#1A1A1E] border p-4
  [Industry badge]     — 업종 12px chip (tag chip bg)
  [BrandNames]         — 대표 브랜드명 2~3개 (16px semibold accent)
  [Divider]            — border-t border-[#4A4440] mt-3
  [Footer row]
    [AuthorLabel]      — 작성자 이메일 앞 2자 + @domain 12px tertiary
    [Actions]          — LikeButton + count | BookmarkButton
```

### GalleryModal 내부 구조

```
[Backdrop]
  [Modal container]  max-w-[720px] bg-[#1A1A1E] rounded-2xl p-8
    [Header row]     — 프로젝트 제목 (20px semibold) + X 버튼
    [Industry chip]  — 업종 chip
    [Section: 입력 요약]  — key-value 그리드 (label 12px tertiary, value 14px secondary)
    [Section: 브랜드명 목록]  — 브랜드명 전체 (RecommendCardItem 패턴 재사용)
    [Section: 피드백]        — LikeButton + BookmarkButton (대형, 44px target)
```

**2. 자동 스크롤 useEffect 추가:**

업종 변경 useEffect (line 42-51) 다음에 새 useEffect 추가:
```typescript
// 모바일 자동 스크롤 -- 추천 완료 시 추천 패널로 이동 (D-02)
useEffect(() => {
  const prevCount = prevBatchCountRef.current;
  prevBatchCountRef.current = batches.length;

  // 새 배치가 추가되었을 때 (로딩 완료 후)
  if (!isLoading && batches.length > prevCount && prevCount > 0) {
    if (window.innerWidth < 1024 && recommendPanelRef.current) {
      setTimeout(() => {
        recommendPanelRef.current?.scrollIntoView({
          behavior: 'smooth',
          block: 'start',
        });
      }, 100);
    }
  }
}, [isLoading, batches.length]);
```

트리거 조건:
- `!isLoading`: 로딩이 완료된 상태
- `batches.length > prevCount`: 새 배치가 실제로 추가됨 (에러 시 스크롤 안 함)
- `prevCount > 0`: 최초 로드 시 스크롤 방지 (첫 추천만 해당)
- `window.innerWidth < 1024`: 모바일에서만 작동

타이밍: 100ms setTimeout으로 DOM 렌더링 대기 (UI-SPEC 명시).

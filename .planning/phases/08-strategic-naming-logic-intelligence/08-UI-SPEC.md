## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, chip 내부 padding, inline spacing |
| sm | 8px | Compact element spacing, 칩 간 간격 |
| md | 16px | Default element spacing, 탭 내부 컴포넌트 간격 |
| lg | 24px | Section padding, 탭 콘텐츠 영역 padding |
| xl | 32px | 탭 간 전환 영역, 위저드-추천패널 간격 |
| 2xl | 48px | Major section breaks (히어로-위저드 사이) |
| 3xl | 64px | Page-level spacing (사용 안 함, 예비) |

Exceptions: 슬라이더 thumb touch target 44px (접근성), 탭 높이 44px (터치 타겟)

## Typography

|------|------|--------|-------------|

| Label | 14px | 600 (semibold) | 1.4 |

## Component Inventory (Phase 8 신규)

### 1. WizardTabs — 탭 네비게이션

```
구조: 상단 고정 탭 바 [분석] [정체성] [표현]
```

| Property | Spec |
|----------|------|
| 높이 | 44px (터치 타겟) |
| 배경 | `#2C2825` |
| 비활성 탭 | text `#A09890`, font 14px semibold |
| 활성 탭 | text `#B48C50`, font 14px semibold |
| Indicator | 하단 2px bar, `#B48C50`, 탭 전체 너비, 탭 전환 시 translateX 트랜지션 300ms ease |
| 간격 | 탭 간 0px (균등 분할, flex-1) |
| Border | 하단 1px `#4A4440` (전체 탭 바) |

상태:
- Default: 3개 탭 균등 분할, 첫 번째 탭(분석) 활성
- 자유 이동: 어떤 탭이든 클릭 시 즉시 전환 (순서 강제 없음)
- 탭 전환 애니메이션: 콘텐츠 fadeIn 200ms, indicator slideX 300ms

## Interaction Contracts

### 탭 전환

| Trigger | Action |
|---------|--------|
| 탭 클릭 | 해당 탭 콘텐츠로 즉시 전환, indicator 슬라이드 |
| 탭 전환 시 | 입력값 유지 (Zustand store), 추천 패널도 해당 단계 데이터로 갱신 |
| 모바일 | 탭 바 sticky top (스크롤 시 상단 고정) |

### 칩 선택

| Trigger | Action |
|---------|--------|
| 칩 클릭 | 선택/해제 토글, 선택 시 scale(0.95) → scale(1) 100ms 피드백 |
| 최대 초과 | 가장 먼저 선택된 항목 자동 해제 + 신규 선택 |
| 접근성 | 각 칩 role="checkbox", aria-checked, keyboard Tab+Space |

### 슬라이더

| Trigger | Action |
|---------|--------|
| 드래그/클릭 | 즉시 값 변경, 숫자 표시 업데이트 |
| 접근성 | input[type=range], aria-label="{키워드} 가중치", 키보드 좌우 화살표 |

### 카드 확장

| Trigger | Action |
|---------|--------|
| 카드 클릭 | 확장/축소 토글 |
| 확장 시 | 점수 바 0→N% 애니메이션, 콘텐츠 fadeIn |
| 축소 시 | 높이 축소 200ms |
| 접근성 | button role, aria-expanded |

### 고급 옵션 토글

| Trigger | Action |
|---------|--------|
| 토글 클릭 | 내부 콘텐츠 slideDown/slideUp 200ms |
| 아이콘 | 열림 시 ChevronUp, 닫힘 시 ChevronDown |

## Layout Contract

```
[InputPanel 70%] | [RecommendPanel 30%]  (기존 비율 유지, lg 이상)

InputPanel 내부:
  [히어로 영역]
  [WizardTabs — sticky]
  [탭 콘텐츠 영역]
    - 분석: StoreBasic → CompetitorInput → USPInput → KeywordWeight → 추천버튼

RecommendPanel 내부:
  [RationaleExpandedCard ...] (확장 가능 카드 리스트)
  or [EmptyState]
```

모바일 (lg 미만): 세로 스택, InputPanel → RecommendPanel 순서 유지. 탭 바 sticky top.
